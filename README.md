# code-sample-task

Recruitment task based on AdobeXD design.

## Requirements

- node: 22.x (20+)
- npm: 10.x

## Commands

Quickstart :

```sh
npm i && npm run dev
```

### Install dependencies

```sh
npm i
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Test in production mode

```sh
npm run preview
```
